learn-gadgetron: A repository for learning gadgetron
====================================================

This documentation provides support for a users first attempts at using gadgetron
by providing scripts for building a singularity image that can be used to run gadgetron
and by providing steps for using this image to complete the 2020 summer tutorial on 
gadgetron that was conducted online.
